# Welcome to Coinpengin-Frontend

![Coin Pengin Preview](https://i.imgur.com/KF0mvRr.png)

Coinpengin is a cryptocurrency tracking platform featuring real-time market monitoring for the top 100 coins, interactive price charts, and secure wishlist management with full user authentication. This repository is currently undergoing a complete architectural overhaul to modernize the legacy codebase into a premium, type-safe, and production-ready experience.

## Project Status: UI Revamp in Progress

The **master branch** contains the original stable version of Coinpengin.

I am currently finalizing a complete architectural and UI overhaul which is available in the **[refactor branch](https://github.com/syliow/coinpengin-frontend/tree/refactor)**. This revamp modernizes the entire stack using an AI-integrated workflow, migrating the codebase to TypeScript and Tailwind CSS for a premium, high-performance experience.

### Key Refactor Improvements:

- **TypeScript Migration**: Full conversion for robust type safety and improved developer experience.
- **Tailwind CSS**: Replaced legacy styling with a modern, utility-first architecture.
- **Modern UI/UX**: Redesigned using Headless UI and Framer Motion for a premium feel.
- **AI-Assisted Efficiency**: Leveraged AI agents to handle bulk transformations while maintaining lead oversight on system logic and design.

## Features

- **Real-Time Tracking**: Monitor cryptocurrency prices in real-time.
- **Advanced Charts**: Interactive price charts powered by Chart.js.
- **Currency Switching**: Seamlessly toggle between USD and MYR.
- **Premium Dark Mode**: Native dark/light mode support via Tailind CSS.
- **User Systems**: Secure account registration and login.
- **Personalized Wishlist**: Add and manage cryptocurrency watchlists.

## Technologies Used

- **React 18** - Modern functional components and hooks.
- **TypeScript** - Industry-standard type safety.
- **Tailwind CSS** - Modern, performant styling.
- **Headless UI** - Unstyled, accessible UI components.
- **Framer Motion** - High-performance animations and transitions.
- **Axios** - Standardized API communication.
- **Chart.js** - Dynamic data visualization.
- **Formik & Yup** - Robust form handling and validation.

## Installation

1. Clone this repo: `git clone https://github.com/syliow/coinpengin-frontend.git`
2. Install dependencies: `npm install`
3. Start the project: `npm run start` (Localhost: 3000)
4. _Note: Requires [Coinpengin Backend](https://github.com/syliow/coinpengin-backend) for user features._
